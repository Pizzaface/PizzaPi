Where do extensions live in the layout?

Extensions are peers to Terminal & File Explorer — but where exactly do they sit?

A

Extend the CombinedPanel

Extensions become additional tabs alongside Terminal and File Explorer in the existing right panel. Simple, familiar, no layout changes needed.

Separate Extension Panel
