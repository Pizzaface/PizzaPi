## Task Management

This project uses Beads for task management. Key commands:
- `bd ready` - Show available work
- `bd create --title="..." --type=task` - Create a task
- `bd show <id>` - View task details
- `bd close <id>` - Complete a task
- `bd sync` - Sync with git
