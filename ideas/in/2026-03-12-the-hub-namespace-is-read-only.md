The Hub namespace is read-only — clients can observe live sessions but can't interact. There's an opportunity for collaborative/spectator features:

- **Live annotation** — viewers can drop comments on agent output in real-time
- **Intervention** — authorized viewers can inject messages or override agent decisions
- **Shared cursors** — see who else is watching a session (Google Docs style)
- **Session forking** — clone a running session's state to experiment without affecting the original
- **Approval gates** — agents can request human approval from any connected viewer, not just the session owner
